一、 基础语法与输入输出 (1-10)
Hello World：打印 "Hello, World!"。
个性化问候：输入名字，打印 "Hello, [名字]!"。
两数相加：输入两个数字，打印它们的和。
数据类型探索：打印不同类型变量（int, float, str, bool）及其 type()。
字符串拼接：输入姓和名，合并并打印全名。
字符转换：输入一段文字，分别打印大写和小写版本。
简单计算器：输入两个数和运算符，打印计算结果。
年龄计算器：输入出生年份，计算并打印当前年龄。
温度转换器：摄氏度与华氏度互相转换。
货币转换器：人民币与美元按固定汇率互相转换。


二、 条件判断与逻辑 (11-20)
奇偶数判断：判断输入的数字是奇数还是偶数。
正负数判断：判断数字是正数、负数还是零。
闰年判断：判断输入的年份是否为闰年。
成绩评级：根据分数输出 A/B/C/D/F 等级。
BMI 计算器：输入身高体重，计算 BMI 并给出健康建议。
门票价格：根据年龄和是否为学生，计算游乐园门票价格。
三角形判断：输入三边长，判断能否构成三角形。
三角形类型：判断是等边、等腰还是普通三角形。
密码验证：检查密码长度是否大于8且包含数字和字母。
简易登录：验证用户名和密码是否匹配预设值。
