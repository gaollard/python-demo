# 元组的组包和解包

组包（packing）是把多个值「装进」一个元组；解包（unpacking）是把元组里的元素「拆开」赋给多个变量。两者经常成对出现，是 Python 里非常常见的写法。

## 组包（Packing）

多个值用逗号连在一起，就会自动组成一个元组：

```python
# 显式括号
point = (10, 20)
print(point)       # (10, 20)
print(type(point)) # <class 'tuple'>

# 省略括号，效果相同
point = 10, 20
print(point)       # (10, 20)
```

函数「返回多个值」本质就是组包：

```python
def get_size():
    return 1920, 1080  # 组包成一个元组返回

result = get_size()
print(result)  # (1920, 1080)
```

## 解包（Unpacking）

左侧变量个数与右侧元素个数一致时，可以一一对应赋值：

```python
point = (10, 20)
x, y = point
print(x, y)  # 10 20

# 也可以直接对字面量解包
width, height = 1920, 1080
print(width, height)  # 1920 1080
```

组包 + 解包连在一起，就是「多返回值」的常见写法：

```python
def get_user():
    return "Alice", 25  # 组包

name, age = get_user()  # 解包
print(name, age)        # Alice 25
```

## 交换变量

利用右侧先组包、左侧再解包，无需临时变量：

```python
a, b = 1, 2
a, b = b, a
print(a, b)  # 2 1
```

## 用 `_` 忽略不需要的值

约定用 `_` 表示「这个值我不要」：

```python
person = ("Bob", 30, "bob@example.com")
name, age, _ = person
print(name, age)  # Bob 30

# 只要第一个
first, *_ = (10, 20, 30, 40)
print(first)  # 10
```

## 星号解包 `*`

元素个数不确定，或只关心首尾时，用 `*` 收集剩余部分（结果是**列表**）：

```python
nums = (1, 2, 3, 4, 5)

first, *middle, last = nums
print(first)   # 1
print(middle)  # [2, 3, 4]  ← 注意是 list
print(last)    # 5

head, *tail = nums
print(head)  # 1
print(tail)  # [2, 3, 4, 5]

*head, tail = nums
print(head)  # [1, 2, 3, 4]
print(tail)  # 5
```

注意：同一个解包表达式里，`*` 只能出现一次。

```python
# a, *b, *c, d = (1, 2, 3, 4)  # SyntaxError
```

## 嵌套解包

元组里还有元组时，可以按结构嵌套解包：

```python
data = ("Alice", (90, 85, 88))
name, (math, english, physics) = data
print(name)     # Alice
print(math)     # 90
print(english)  # 85
```

## 遍历时解包

常用于「元组组成的序列」：

```python
points = [(0, 0), (1, 2), (3, 4)]

for x, y in points:
    print(x, y)

# 带索引
for i, (x, y) in enumerate(points):
    print(i, x, y)
```

## 常见错误

```python
# 1. 数量不匹配
# a, b = (1, 2, 3)   # ValueError: too many values to unpack
# a, b, c = (1, 2)   # ValueError: not enough values to unpack

# 2. 单元素元组别漏逗号
t = (1,)
# x, = t   # 正确：解包得到 x = 1
# x = t    # 这是整个元组，不是解包
```

不确定个数时，用 `*` 兜底：

```python
a, *rest = (1, 2, 3, 4)
print(a, rest)  # 1 [2, 3, 4]
```

## 小结

| 概念 | 含义 | 例子 |
|---|---|---|
| 组包 | 多个值 → 一个元组 | `t = 1, 2` / `return a, b` |
| 解包 | 一个元组 → 多个变量 | `x, y = t` |
| `*` | 收集剩余元素为列表 | `a, *rest = t` |
| `_` | 忽略不需要的值 | `name, _, email = t` |

- 组包靠**逗号**，括号常可省略
- 解包时左右数量要对齐；不定长用 `*`
- 交换变量、函数多返回值，都是组包 + 解包的组合
