# 集合（Set）

集合是**无序、不重复**的元素容器。适合去重、成员判断，以及做交并差等集合运算。

## 特点一览

| | 集合 `set` | 列表 `list` | 元组 `tuple` |
|---|---|---|---|
| 写法 | `{1, 2, 3}` | `[1, 2, 3]` | `(1, 2, 3)` |
| 有序 | 否（无下标） | 是 | 是 |
| 可重复 | 否（自动去重） | 是 | 是 |
| 可变性 | 可变 | 可变 | 不可变 |
| 元素要求 | 必须可哈希 | 任意 | 任意 |
| 典型用途 | 去重、成员检测、集合运算 | 有序可变序列 | 固定结构 |

## 创建

```python
# 花括号字面量
s = {1, 2, 3}
print(s)         # {1, 2, 3}
print(type(s))   # <class 'set'>

# 空集合必须用 set()，{} 是空字典
empty = set()
print(type(empty))  # <class 'set'>
print(type({}))     # <class 'dict'>

# 从其他可迭代对象创建（自动去重）
s1 = set([1, 2, 2, 3])
s2 = set("hello")
print(s1)  # {1, 2, 3}
print(s2)  # {'h', 'e', 'l', 'o'}
```

元素必须可哈希（immutable）：数字、字符串、元组可以；列表、字典、集合本身不行。

```python
ok = {1, "a", (1, 2)}
# bad = {[1, 2]}   # TypeError: unhashable type: 'list'
```

## 增删改

```python
s = {1, 2, 3}

# 添加单个元素
s.add(4)
print(s)  # {1, 2, 3, 4}

# 批量添加（可传列表、元组、集合等）
s.update([5, 6], {7})
print(s)  # {1, 2, 3, 4, 5, 6, 7}

# 删除：元素不存在时 remove 报错，discard 静默
s.remove(7)
s.discard(99)   # 没有 99 也不报错
# s.remove(99)  # KeyError

# 随机弹出一个元素
x = s.pop()
print(x, s)

# 清空
s.clear()
print(s)  # set()
```

集合无下标，不能 `s[0]` 访问或赋值。

## 成员判断

集合的 `in` / `not in` 平均时间复杂度 O(1)，比列表快很多：

```python
s = {"apple", "banana", "cherry"}

print("apple" in s)      # True
print("orange" not in s) # True
```

## 遍历

```python
s = {3, 1, 2}
for item in s:
    print(item)  # 顺序不保证，可能是 1 2 3 或其他

# 需要有序时先转 sorted
for item in sorted(s):
    print(item)  # 1 2 3
```

## 集合运算

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# 并集：属于 a 或属于 b
print(a | b)           # {1, 2, 3, 4, 5, 6}
print(a.union(b))      # 同上

# 交集：同时属于 a 和 b
print(a & b)           # {3, 4}
print(a.intersection(b))

# 差集：在 a 中但不在 b 中
print(a - b)           # {1, 2}
print(a.difference(b))

# 对称差：只在其中一个集合里（并集减交集）
print(a ^ b)           # {1, 2, 5, 6}
print(a.symmetric_difference(b))
```

就地修改（改的是 a 本身）：

```python
a = {1, 2, 3}
b = {3, 4}

a |= b   # a.update(b)，并集写回 a
# a &= b   # 交集
# a -= b   # 差集
# a ^= b   # 对称差
print(a)
```

## 子集 / 超集 / 相等

```python
a = {1, 2}
b = {1, 2, 3}
c = {1, 2}

print(a < b)    # True，真子集
print(a <= b)   # True，子集（含相等）
print(b > a)    # True，真超集
print(a == c)   # True

print(a.issubset(b))    # True
print(b.issuperset(a))  # True
print(a.isdisjoint({9, 10}))  # True，无交集
```

## 不可变集合 `frozenset`

`frozenset` 不可修改，因此可哈希，能作为字典的键或放进另一个集合：

```python
fs = frozenset([1, 2, 3])
# fs.add(4)  # AttributeError

d = {fs: "ok"}
print(d[fs])  # ok

nested = {frozenset([1, 2]), frozenset([3, 4])}
```

集合运算对 `frozenset` 同样适用；运算结果类型取决于左侧操作数。

## 常见用途

**1. 去重（顺序不保留）**

```python
nums = [1, 2, 2, 3, 3, 3]
unique = list(set(nums))
print(unique)  # 如 [1, 2, 3]，顺序不保证

# 需要保序去重（Python 3.7+ dict 保序）
unique_ordered = list(dict.fromkeys(nums))
print(unique_ordered)  # [1, 2, 3]
```

**2. 快速判断是否存在**

```python
allowed = {"admin", "editor", "viewer"}
role = "editor"
if role in allowed:
    print("有权限")
```

**3. 找两个列表的公共 / 差异元素**

```python
users_a = ["Alice", "Bob", "Carol"]
users_b = ["Bob", "Dave"]

common = set(users_a) & set(users_b)
only_a = set(users_a) - set(users_b)
print(common)  # {'Bob'}
print(only_a)  # {'Alice', 'Carol'}
```

## 集合推导式

```python
squares = {x * x for x in range(6)}
print(squares)  # {0, 1, 4, 9, 16, 25}

evens = {x for x in range(10) if x % 2 == 0}
print(evens)  # {0, 2, 4, 6, 8}
```

## 与列表 / 字典的转换

```python
lst = [1, 2, 2, 3]
s = set(lst)       # {1, 2, 3}
back = list(s)     # [1, 2, 3]（顺序不保证）

# 字典的键可以变成集合
d = {"a": 1, "b": 2}
print(set(d))      # {'a', 'b'}
```

## 小结

- 需要**去重 / 成员检测 / 交并差** → 用集合
- 空集合写 `set()`，不要写 `{}`
- 元素必须可哈希；要嵌套集合用 `frozenset`
- 无序、无下标；要有序输出用 `sorted(s)`
- `in` 很快；`remove` 不存在会报错，`discard` 不会
