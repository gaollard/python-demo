Python 的 for 循环是遍历可迭代对象（如列表、字符串、字典等）的核心工具。它采用“for 变量 in 可迭代对象”的语法，自动处理迭代过程，无需手动管理索引，代码简洁且意图清晰。

📝 基础语法与常用场景

for 循环的基本结构如下：

for 变量 in 可迭代对象:
    # 循环体：对每个元素执行的代码（注意4个空格缩进）

以下是几种最常见的用法：

遍历列表/元组
直接获取序列中的每个元素。
fruits = ["苹果", "香蕉", "橙子"]
for fruit in fruits:
    print(f"我喜欢吃{fruit}")

遍历字符串
将字符串视为字符序列，逐个访问。
word = "Python"
for char in word:
    print(char)

按固定次数循环
使用 range() 函数生成数字序列，是执行固定次数循环的最佳实践。
range(5) 生成 0, 1, 2, 3, 4
for i in range(5):
    print(f"循环第 {i+1} 次")

range(起始, 结束, 步长) 生成 1, 3, 5, 7, 9
for i in range(1, 10, 2):
    print(i)

遍历字典
字典提供了三种遍历方式，其中 .items() 最为常用。
student = {"姓名": "小明", "年龄": 18}

遍历键值对（最常用）
for key, value in student.items():
    print(f"{key}: {value}")

仅遍历键
for key in student:
    print(key)

仅遍历值
for value in student.values():
    print(value)

🛠️ 进阶用法与最佳实践

同时获取索引和元素
当需要知道当前元素的索引时，优先使用 enumerate()，而不是通过 range(len(list)) 手动索引，这样更安全、可读性更高。
fruits = ["苹果", "香蕉", "橙子"]
start=1 让索引从1开始，更符合日常计数习惯
for index, fruit in enumerate(fruits, start=1):
    print(f"{index}. {fruit}")

⚠️ 重要注意事项

避免在循环中修改序列
    在遍历列表时直接增删元素可能导致意外行为或跳过元素。建议使用列表推导式创建新列表，或遍历原列表的副本。
        # ❌ 错误做法
    numbers = 
    for num in numbers:
        if num % 2 == 0:
            numbers.remove(num)

    # ✅ 推荐做法：使用列表推导式
    numbers = 
    numbers = [num for num in numbers if num % 2 != 0]
    

range() 是左闭右开区间
    range(1, 5) 生成的是 1, 2, 3, 4，不包含结束值 5。这是初学者常犯的错误。

range() 是惰性求值
    range() 不会一次性生成所有数字，而是按需产生，在处理大范围循环时内存效率极高。

嵌套循环处理多维数据
