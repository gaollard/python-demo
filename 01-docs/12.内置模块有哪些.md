# 内置模块有哪些

Python 安装后自带一套**标准库**（Standard Library），其中的模块称为内置模块 / 标准库模块。  
它们**无需 pip 安装**，使用前需要 `import`。

> 注意：内置模块 ≠ 内置函数。  
> - 内置函数：`print`、`len`、`open` —— 不用导入  
> - 内置模块：`os`、`math`、`json` —— 必须 `import`

---

## 常用内置模块（按用途）

### 1. 系统与运行环境

| 模块 | 作用 |
|------|------|
| `os` | 文件/目录、环境变量、进程 |
| `sys` | 命令行参数、解释器信息、退出程序 |
| `pathlib` | 面向对象的路径操作（推荐） |
| `shutil` | 高级文件复制、移动、删除 |
| `subprocess` | 启动外部命令/子进程 |

```python
import os
import sys

print(os.getcwd())      # 当前工作目录
print(sys.argv)         # 命令行参数列表
```

### 2. 时间与日期

| 模块 | 作用 |
|------|------|
| `time` | 时间戳、休眠 `sleep`、简单计时 |
| `datetime` | 日期时间对象、格式化、加减 |
| `calendar` | 日历相关 |

```python
from datetime import datetime, timedelta

now = datetime.now()
print(now.strftime("%Y-%m-%d %H:%M:%S"))
print(now + timedelta(days=7))
```

### 3. 数学与随机

| 模块 | 作用 |
|------|------|
| `math` | 三角函数、对数、`pi`、`sqrt` 等 |
| `random` | 随机数、抽样、打乱序列 |
| `statistics` | 均值、中位数、方差等 |
| `decimal` | 精确十进制运算 |
| `fractions` | 分数运算 |

```python
import math
import random

print(math.sqrt(16), math.pi)
print(random.randint(1, 100))
print(random.choice(["a", "b", "c"]))
```

### 4. 字符串与数据格式

| 模块 | 作用 |
|------|------|
| `re` | 正则表达式匹配/替换 |
| `json` | JSON 序列化 / 反序列化 |
| `csv` | 读写 CSV |
| `base64` | Base64 编解码 |
| `hashlib` | MD5、SHA 等哈希 |
| `uuid` | 生成唯一 ID |

```python
import json
import re

data = {"name": "Tom", "age": 18}
s = json.dumps(data, ensure_ascii=False)
print(json.loads(s))

print(re.findall(r"\d+", "订单号 12345，金额 99"))
```

### 5. 数据结构与迭代工具

| 模块 | 作用 |
|------|------|
| `collections` | `Counter`、`defaultdict`、`deque`、`namedtuple` |
| `itertools` | 排列组合、无限迭代、链式拼接 |
| `functools` | `partial`、`lru_cache`、`reduce` |
| `copy` | 浅拷贝 / 深拷贝 |
| `enum` | 枚举类型 |
| `typing` | 类型注解辅助 |

```python
from collections import Counter, defaultdict

print(Counter("hello"))                 # 计数
d = defaultdict(list)
d["fruit"].append("apple")
```

### 6. 文件、压缩与序列化

| 模块 | 作用 |
|------|------|
| `io` | 流、内存文件 |
| `pickle` | Python 对象序列化 |
| `gzip` / `zipfile` / `tarfile` | 压缩与归档 |
| `tempfile` | 临时文件/目录 |
| `configparser` | 读写 ini 配置 |

### 7. 网络与并发

| 模块 | 作用 |
|------|------|
| `socket` | TCP/UDP 套接字 |
| `http` | HTTP 服务端/客户端相关 |
| `urllib` | URL 打开与解析 |
| `email` | 邮件处理 |
| `threading` | 多线程 |
| `multiprocessing` | 多进程 |
| `asyncio` | 异步 IO |
| `queue` | 线程安全队列 |
| `concurrent.futures` | 线程池 / 进程池 |

```python
import socket

s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
# s.connect(("127.0.0.1", 8080))
```

### 8. 日志、调试与测试

| 模块 | 作用 |
|------|------|
| `logging` | 分级日志 |
| `traceback` | 异常堆栈 |
| `pdb` | 调试器 |
| `unittest` | 单元测试 |
| `doctest` | 文档中的示例测试 |

```python
import logging

logging.basicConfig(level=logging.INFO)
logging.info("程序启动")
```

## 怎么查看本机有哪些模块

```python
# 交互环境中查看（输出很长）
help("modules")

# 查看某个模块的用法
import math
help(math)
dir(math)   # 列出模块里有什么
```

官方完整清单：[Python 标准库参考](https://docs.python.org/zh-cn/3/library/index.html)

## 和「第三方模块」的区别

| 类型 | 来源 | 安装 | 示例 |
|------|------|------|------|
| 内置 / 标准库 | Python 自带 | 不用装 | `os`、`json`、`math` |
| 第三方库 | PyPI 等 | `pip install` | `requests`、`numpy`、`django` |
| 自定义模块 | 自己写的 `.py` | — | `circle_func.py` |

---

## 小结

1. 内置模块 = 标准库模块，随 Python 安装，需 `import` 后使用。  
2. 日常最高频：`os` / `pathlib`、`sys`、`datetime`、`json`、`re`、`random`、`math`、`collections`。  
3. 需要更多能力时，先查标准库，再考虑第三方包。
