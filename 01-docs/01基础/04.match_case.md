在 Python 3.10+ 中，`match` / `case` 是结构化模式匹配（Structural Pattern Matching）。它类似其他语言的 `switch`，但更强：除了比较值，还能按结构拆解数据。

🔍 基本语法

```python
match 表达式:
    case 模式1:
        处理逻辑
    case 模式2:
        处理逻辑
    case _:
        默认分支（类似 else）
```

`_` 是通配符，匹配任意值，通常放在最后作为默认分支。

📦 字面量匹配

最简单的用法：按具体值分支，适合替代一长串 `if-elif`。

```python
status = 404

match status:
    case 200:
        print("成功")
    case 404:
        print("未找到")
    case 500:
        print("服务器错误")
    case _:
        print("未知状态码")
```

🔗 多值匹配（或模式）

用 `|` 表示“或”，多个值走同一分支。

```python
day = "Sat"

match day:
    case "Mon" | "Tue" | "Wed" | "Thu" | "Fri":
        print("工作日")
    case "Sat" | "Sun":
        print("周末")
    case _:
        print("无效日期")
```

📥 捕获变量

模式里可以直接绑定变量，匹配成功后可在分支内使用。

```python
point = (3, 5)

match point:
    case (0, 0):
        print("原点")
    case (x, 0):
        print(f"在 X 轴上，x = {x}")
    case (0, y):
        print(f"在 Y 轴上，y = {y}")
    case (x, y):
        print(f"点坐标: ({x}, {y})")
```

🛡️ 守卫条件（if 守卫）

在 `case` 后加 `if`，只有模式匹配且条件为真时才进入该分支。

```python
age = 20

match age:
    case n if n < 18:
        print("未成年")
    case n if 18 <= n < 65:
        print("成年人")
    case n if n >= 65:
        print("老年人")
```

📋 序列与映射匹配

可以按列表/元组长度和内容匹配，也可按字典的键值结构匹配。

```python
# 序列
command = ["go", "north"]

match command:
    case ["quit"]:
        print("退出")
    case ["go", direction]:
        print(f"向 {direction} 走")
    case ["drop", *items]:
        print(f"丢弃: {items}")
    case _:
        print("未知命令")

# 映射
user = {"name": "Alice", "age": 20}

match user:
    case {"name": name, "age": age} if age >= 18:
        print(f"{name} 已成年")
    case {"name": name}:
        print(f"用户: {name}")
```

⚡ 与 if-elif 对比

- 简单布尔条件、范围判断：继续用 `if` / `elif` 更直观。
- 多分支按值/结构分流：`match` / `case` 更清晰，尤其适合状态码、命令解析、数据结构拆解。
- `match` 按从上到下匹配，命中第一个成功的 `case` 后不再继续。
- 需要 Python 3.10 及以上版本。
