`del` 是 Python 的语句（不是函数），用来删除名字绑定，或删除容器中的元素。

核心要记住：它删的是「名字」或「容器里的一项」，不是立刻销毁对象本身。对象是否被回收，取决于还有没有别的引用指向它。

📌 基本用法

删除变量名

x = 10
del x
# print(x)  # NameError: name 'x' is not defined

删除后，名字 `x` 从当前作用域消失。如果原先那个对象没有其他引用，垃圾回收才会真正释放它。

删除列表元素（按索引）

bicycles = ['trek', 'cannondale', 'redline', 'specialized']
del bicycles[0]
print(bicycles)  # ['cannondale', 'redline', 'specialized']

删除字典键

alien_0 = {'color': 'green', 'points': 5}
del alien_0['points']
print(alien_0)  # {'color': 'green'}

删除切片

nums = [0, 1, 2, 3, 4, 5]
del nums[1:4]
print(nums)  # [0, 4, 5]

也可以清空整个列表：`del nums[:]`（列表对象还在，只是变空了）。

🔍 del 到底做了什么？

Python 里名字和对象是分开的：

a = [1, 2, 3]
b = a          # a 和 b 指向同一个列表
del a          # 只删掉名字 a
print(b)       # [1, 2, 3]  列表还在

所以：

del 名字 → 解除「名字 → 对象」的绑定
del 容器[索引/键] → 从容器里移除该项
对象销毁 → 由引用计数 / 垃圾回收决定，不是 del 立刻完成的

⚖️ del vs 其他删除方式

场景   常用写法   特点
按索引删列表元素   del lst[i]   无返回值；索引不存在会 IndexError
按索引删并拿到值   lst.pop(i)   返回被删元素；默认删最后一个
按值删列表元素   lst.remove(x)   删第一个等于 x 的元素；找不到会 ValueError
清空列表   lst.clear() 或 del lst[:]   列表对象还在，内容清空
删字典键   del d[k]   键不存在会 KeyError
删字典键（安全）   d.pop(k, None)   可给默认值，避免 KeyError
删变量名   del name   名字从作用域移除

简单对照：

lst = [10, 20, 30]

del lst[1]        # lst → [10, 30]，没有返回值
x = lst.pop(0)    # x = 10，lst → [30]
lst.remove(30)    # lst → []

⚠️ 常见陷阱

1. del 不是函数

# 错误写法
del(x)     # 看起来像函数调用，其实只是 del 后面的括号表达式
# 正确写法
del x

`del(x)` 能跑是因为 `(x)` 只是加了括号的名字，不是函数调用。习惯上写成 `del x`。

2. 字典键不存在会报错

d = {'a': 1}
del d['b']   # KeyError: 'b'

更稳妥：

d.pop('b', None)

3. 遍历时用 del 删元素很危险

lst = [1, 2, 3, 4]
for i, v in enumerate(lst):
    if v % 2 == 0:
        del lst[i]   # ❌ 边遍历边删，索引会错乱

正确做法通常是建新列表，或倒序删除：

lst = [v for v in lst if v % 2 != 0]

4. del 属性

class User:
    def __init__(self):
        self.name = 'Alice'

u = User()
del u.name
# print(u.name)  # AttributeError

也可以删类属性、模块属性等，本质都是解除绑定。

5. 切片删除与赋值清空

nums = [1, 2, 3]
del nums[:]     # 清空内容，nums 还是原来那个列表对象
nums = []       # 名字 nums 指向一个全新的空列表

如果别处还引用着旧列表，这两种写法效果不同。

📦 还能删什么？

# 元组不能删元素（不可变）
t = (1, 2, 3)
# del t[0]      # TypeError

# 但可以删整个名字
del t

# 多目标一起删
x, y = 1, 2
del x, y

# 删切片、步长切片
a = [0, 1, 2, 3, 4, 5]
del a[::2]      # 删掉偶数下标 → [1, 3, 5]

📌 一句话总结

`del` 用来解除名字绑定，或从可变容器中移除元素；它不保证立刻销毁对象。删列表用索引，删字典用键，需要返回值时优先用 `pop`，按值删列表元素用 `remove`。
