💡 重要概念澄清

需要特别注意的是，Python 的“内置模块”与“内置函数/类型”是两个不同的概念：

内置模块：指标准库中的模块（如 os, json），需要通过 import 语句导入后才能使用。
内置函数/类型：指无需导入即可直接使用的函数和类型，例如 print(), len(), int, list, dict 等。

🔍 如何查找更多模块

Python 标准库非常庞大，以上仅列出了常用部分。要探索完整的模块列表，建议：

查阅官方文档：Python 官方文档的“标准库参考”章节提供了所有内置模块的完整清单和详细说明。
使用 help() 函数：在 Python 交互式解释器中，输入 help('modules') 可以列出当前环境中所有可用的模块。
查看模块路径：标准库模块通常存放在 Python 安装目录的 lib 文件夹下，可以直接浏览该目录了解模块结构。

提示：标准库是 Python 开发者的核心工具集，熟练掌握常用模块能显著提升开发效率，避免重复造轮子。建议根据实际项目需求，有针对性地学习和使用相关模块。
