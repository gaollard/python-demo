Python 的 random 模块是生成伪随机数的标准库，基于 Mersenne Twister 算法，适用于模拟、游戏、数据采样等场景。

⚠️ 安全警告
random 模块生成的随机数是完全确定性的，绝对不适用于密码学、安全令牌、验证码等安全敏感场景。 如需安全随机数，请使用 secrets 模块。

🔢 生成随机数
函数   功能   示例
random()   生成 [0.0, 1.0) 的随机浮点数   random.random()

uniform(a, b)   生成 [a, b] 的随机浮点数   random.uniform(1.5, 5.5)

randint(a, b)   生成 [a, b] 的随机整数（含两端）   random.randint(1, 10)

randrange(start, stop, step)   生成 [start, stop) 内步长为 step 的随机整数   random.randrange(0, 100, 5)

| getrandbits(k) | 生成 k 比特长的随机整数 | random.getrandbits(16) |

🎲 序列随机操作
函数   功能   示例
choice(seq)   从非空序列中随机选一个元素   random.choice(['A', 'B', 'C'])

sample(population, k)   从序列中随机选 k 个不重复元素   random.sample(, 3)

| shuffle(x) | 原地打乱序列顺序，无返回值 | random.shuffle(my_list) |

🌱 控制随机性：种子

random.seed(a) 用于初始化随机数生成器的内部状态。
不传参或传 None：使用系统时间或操作系统随机源，每次运行结果不同。
传固定值：每次运行生成完全相同的随机数序列，适用于需要可复现结果的场景（如科学实验、调试）。

import random

设置种子，确保结果可复现
random.seed(42)
print(random.randint(1, 100))  # 每次运行都输出相同结果

不设置种子，每次运行结果不同
print(random.randint(1, 100))

📊 随机分布函数

除了均匀分布，random 模块还支持多种统计分布：
函数   分布类型   参数说明
gauss(mu, sigma)   正态（高斯）分布   mu=均值, sigma=标准差

expovariate(lambd)   指数分布   lambd=1/均值

betavariate(alpha, beta)   贝塔分布   alpha, beta > 0

| lognormvariate(mu, sigma) | 对数正态分布 | mu, sigma 为底层正态分布参数 |

💡 实战技巧

生成随机字符串：结合 string 模块和 choice() 可快速生成随机密码或验证码。
        import random
    import string
    chars = string.ascii_letters + string.digits
    password = ''.join(random.choice(chars) for _ in range(12))
    
模拟概率事件：choice() 非常适合模拟抛硬币、抽奖等场景。
        coin = random.choice(['正面', '反面'])
    
无放回抽样：sample() 比手动循环去重更高效，且能保证结果不重复，适合抽奖、分组等场景。
线程安全：全局 random 实例是线程安全的，但在高并发场景下，建议为每个线程创建独立的 random.Random() 实例以避免性能瓶颈。

🔄 与 secrets 模块的对比
特性   random   secrets
用途   模拟、游戏、采样   密码、令牌、密钥

随机性   伪随机（可预测）   密码学安全随机

性能   快   较慢

可复现性   支持 seed()   不支持

掌握 random 模块的这些功能，能让你在数据处理、模拟仿真等场景中游刃有余。如需安全随机数，请务必切换到 secrets 模块。
